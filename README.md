# Análise Global de Energia: Transição e Sustentabilidade

Este repositório contém uma análise visual detalhada sobre o panorama energético mundial, utilizando o dataset **Our World in Data (OWID) Energy Data**. O projeto foi desenvolvido no **Tableau** e foca-se na evolução do consumo e na ascensão das energias renováveis.

Link para o Tableau Public: https://public.tableau.com/shared/2F5Q9XPS6?:display_count=n&:origin=viz_share_link

---

## Objetivos da Visualização

O projeto está dividido em três Dashboards principais, cada um com uma mensagem central:

### 1. Fósseis vs Renováveis
*   **Objetivo:** Mostrar a evolução de energias Fósseis vs Renováveis. ao longo das décadas.

### 2. Distribuição da Participação de Energias Renováveis na Energia Primária 
*   **Objetivo:** Analisar que países têm maior participação de Energias Renováveis na Energia Primária.

### 3. Países com Maior Crescimento em Energias Renováveis
*   **Objetivo:** Países com Maior Crescimento em Energias Renováveis com ênfase no Séc. XXI.
